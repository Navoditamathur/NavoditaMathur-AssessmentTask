# Markdown Meeting Notes → Google Docs (Colab)

Convert markdown meeting notes into a **well-formatted Google Doc** using Python in **Google Colab** and the **Google Docs API**.

The notebook:

- Authenticates to your Google account from Colab
- Parses the meeting notes **line-by-line**
- Creates a new Google Doc
- Applies:
  - Heading styles (H1/H2/H3)
  - Nested bullet lists
  - Different bullet glyphs for nested levels (e.g. disc → diamond → arrow)
  - Google Docs checkboxes for action items (`- [ ]`)
  - Highlighted assignee mentions (e.g. `@sarah`)
  - Footer styling for meeting metadata
